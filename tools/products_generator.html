Генератор products.json

Заполни форму — получи готовый блок кода для вставки в файл конфигурации. Не нужно писать JSON вручную.

📖 Как пользоваться
1
Открой страницу товара в браузере (Wildberries, Ozon или Яндекс.Маркет). Скопируй ссылку из адресной строки.
2
Вставь ссылку в поле ниже — инструмент автоматически определит маркетплейс.
3
Заполни название, установи порог уведомления. Нажми «Сгенерировать».
4
Нажми «Копировать». Открой config/products.json в репозитории → карандаш ✏️ → вставь блок в массив "products": [...] через запятую → Commit.
